# MySQL 面试模拟实战实验室

本目录包含按 SPEC 生成的完整文档：

1. `00.建表脚本.md`：统一建表脚本与设计说明，负责创建五张实验表和基础索引。
2. `01.造数脚本.md`：Python 批量造数说明，真实脚本位于 `../01.scripts/`，包含热点租户、热点用户、热点商品、固定订单。
3. `02.索引实验题.md`：IDX-01 到 IDX-08。
4. `03.锁实验题.md`：LOCK-01 到 LOCK-06。
5. `04.事务与 MVCC 实验题.md`：TX-01 到 TX-06。
6. `05.慢查询排查实验题.md`：SLOW-01 到 SLOW-06。
7. `06.标准答案与观测指标.md`：统一答案、指标解释和面试表达。

建议执行顺序：先执行 `00.建表脚本.md` 创建表结构，再进入 `../01.scripts/` 执行 `./run_seed.sh` 造数，最后依次练习索引、锁、事务、慢查询，并对照标准答案复盘。

说明：`../01.scripts/seed_interview_lab.py` 不负责创建数据库和表结构。请先手动执行 `00.建表脚本.md`，再执行造数脚本。
